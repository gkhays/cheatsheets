# Remote Desktop Cheat Sheet

### Windows Security

```
CTRL + ALT + END
```

### Task Switcher

When _not_ running in full screen use:

```
ALT + PAGE UP
```

### Print Screen

When _not_ running in full screen use:

```
CTRL + ALT + PLUS SIGN (+)
```
