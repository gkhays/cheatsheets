# Remote Desktop Cheat Sheet

### Windows Security

Sometimes called the [Secure attention key](https://en.wikipedia.org/wiki/Secure_attention_key).

```
CTRL + ALT + END
```

### Task Switcher

When _not_ running in full screen use:

```
ALT + PAGE UP
```

### Print Screen

When _not_ running in full screen use:

```
CTRL + ALT + PLUS SIGN (+)
```
