# Gmail Cheat Sheet

### Show keyboard shortcuts
```
Shift + /
```

### Go to search
```
/
```

### Navigate between Priority, Promotions, Social, and Updates tabs
```
~
```

### Navigate through messages
```
j - down
k - up
```

### Select message
```
x
```

### Mark message as read
```
Shift + i
```

### Archive message
```
e
```

### Read message
```
Enter
```

### Delete message
```
#
```

### Go back up to inbox
```
u
```

### Go to inbox
```
g + i
```

### Select all conversations
```
* + a
```

### Move focus to toolbar
```
.
```

### Go to next page
```
g + n
```

### Go to previous page
```
g + p
```
