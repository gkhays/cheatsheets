# VIM Cheat Sheet

### Go to the first line
```
gg
:1
1G
1gg
```

### Cursor movement

`H` - Move to top of screen<br/>
`M` - Move to middle of screen<br/>
`L` - Move to bottom of screen<br/>

### Editing

`J` - join line below<br/>
`cc` - replace entire line<br/>
`C` - replace to end of line<br/>
`c$` - replace to end of line<br/>
`u` - undo<br/>

### Sorting

Sort rows.

```
:%!sort
```
