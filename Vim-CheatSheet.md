# VIM Cheat Sheet

### Go to the first line
```
gg
:1
1G
1gg
```

### Cursor movement

`H` - Move to top of screen
`M` - Move to middle of screen
`L` - Move to bottom of screen

### Editing

`J` - join line below
`cc` - replace entire line
`C` - replace to end of line
`c$` - replace to end of line
`u` - undo
