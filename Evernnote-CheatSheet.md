# Evernote Cheat Sheet

### Format Code Block

```
Ctrl + Shift + L
```

Alternatively, use auto-formatting by typing:

````
``` + Enter
````

### Create Link

With the link in your clipboard, highlight the target text.

```
Ctrl + K Then Ctrl + V
```
