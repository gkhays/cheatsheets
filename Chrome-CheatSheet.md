# Google Chrome Cheat Sheet

### Search Open Tabs
```
Ctrl + Shift + A
```
### Next Tab
Windows
```
Ctrrl + Tab
```
Mac
```
Cmd + Option + Right Arrow
```
### Previous Tab
Windows
```
Ctrl + Shift + Tab
```
Mac
```
Cmd + Option + Left Arrow
```
### Close Current Tab
Windows
```
Ctrl + W
```
Mac
```
Cmd + W
```
